Detailed Description

Pogostuck Engine

Source code for the Pogostuck/Pogo game engine.

Structure

  • components/ - Component classes for game objects
  • scenes/ - Scene management classes for different game states
  • scripts/ - Game logic scripts for player, pogo stick, camera, etc.
  • editor/ - World editor and level serialization code

Building / Running

Just run 'dub' from the source folder like the last PSET. Also might need to run 'brew install sdl3_ttf' (on Mac) or the system equivalent if the font isn't showing up. Also, change the path for font in gameapplication.d. We will make this agnostic at some point.
